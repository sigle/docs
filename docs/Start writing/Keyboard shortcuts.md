Sigle provide the following keyboard shortcuts to get more done in less time:

`Cmd + k` or `Ctrl + k` => link

`Cmd + b` or `Ctrl + b` => bold

`Cmd + i` or `Ctrl + i` => italic

`Cmd + u` or `Ctrl + u` => underlined

` Cmd + \`` or  `Ctrl + \`` => code
